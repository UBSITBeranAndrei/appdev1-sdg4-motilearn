# MotiLearn 🎓

## SDG 4: Quality Education

### Rationale
MotiLearn is a web application aligned with UN Sustainable Development Goal 4 — Quality Education. The app aims to provide motivational learning resources, goal tracking, and educational content to help students stay engaged and inspired in their learning journey.

## Group Members
- Beran, Andrei

## Project Description
MotiLearn is a single-page web application built with Angular. It serves as a platform where learners can access educational motivation tools, track their learning goals, and explore curated resources — all in support of accessible and quality education for everyone.

## Tech Stack
- Angular v21
- TypeScript
- CSS
